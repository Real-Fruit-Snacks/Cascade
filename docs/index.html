01Premise

A vault-based editor over plain markdown.

Cascade is a vault-based markdown editor built on Tauri v2 with a Rust backend and a React + CodeMirror 6 frontend. Three view modes, wiki-links with autocomplete and backlinks, an infinite canvas whiteboard (Obsidian-compatible .canvas format), and real-time collaboration via Yjs CRDT sync.

Files live as plain markdown on your disk. Plugins run in sandboxed iframes with a restrictive CSP and access the vault through message passing.

03Quickstart

Pre-built or from source.

Pre-built installers ship for Windows (.msi), macOS (.dmg), and Linux (.AppImage / .deb). To build from source you need Node 18+, Rust 1.70+, and Tauri CLI 2.x.

# Pre-built — Releases page
# https://github.com/Real-Fruit-Snacks/Cascade/releases

# Build from source
$ git clone https://github.com/Real-Fruit-Snacks/Cascade.git
$ cd Cascade
$ npm install
$ npm run tauri dev       # development
$ npm run tauri build     # production

# Tests + lint
$ npm test
$ npx playwright test
$ npm run lint

05Architecture

Tauri shell, Rust backend.

Tauri v2 provides the native runtime. Rust backend handles file I/O, collab, search. React + CodeMirror 6 frontend with Zustand state. Yjs powers CRDT collab.

src/                 // React frontend
  components/        // 40+ UI components
  editor/            // CodeMirror 6 extensions
    wiki-links.ts
    collab-extension.ts
    slash-commands/
    live-preview/
  stores/            // Zustand state management
  hooks/             // Custom React hooks
  i18n/              // 2200+ translation keys

src-tauri/src/       // Rust backend (Tauri v2)
  collab/            // WebSocket collaboration server
  vault/             // File I/O · themes · plugins · export
  search.rs          // Full-text search
  watcher.rs         // File system watcher

tests/e2e/           // Playwright E2E tests
06Closing

Files on your disk, no lock-in.

Cascade reads and writes plain markdown. No proprietary database. Vault operations go through the Rust backend; the frontend only sees what's needed.

Collab is opt-in per session and ends with the connection. Plugins are sandboxed. Deep links use the cascade:// scheme.

An editor that respects the format it edits.
