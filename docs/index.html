Editor Features

A complete writing environment with every tool you need. Slash commands, focus mode, split panes, and deep customization.

Slash Commands

Type / to insert headings, code blocks, callouts, tables, and more. Context-aware completions with keyboard navigation.

Focus Mode

Paragraph dimming reduces distraction. Typewriter mode keeps the current line centered. Word count goals with progress tracking.

Split Panes

Side-by-side editing with independent scroll. Compare notes, reference while writing, or preview alongside source.

Find & Replace

Search across your entire vault with Ctrl+Shift+F. In-file find and replace with regex support.

Table Editor

Visual table editing with column/row manipulation. Tab through cells. Alignment controls. No raw markdown wrangling.

Smart Lists

Auto-continue bullets, tasks, and numbered lists. Indent/outdent with Tab/Shift+Tab. Checkbox toggling.

Spellcheck

Built-in spellcheck with custom dictionary support. Add words per-vault. No external service required.

Import & Export

Import from Obsidian, Notion, Bear, Roam Research. Export to Markdown, HTML, or PDF with batch support.

Plugin System

Sandboxed iframe execution with restrictive CSP. Plugin marketplace. 30+ configurable feature toggles per vault.

Platform Capabilities

Full-featured on all three desktop platforms. Native performance via Tauri — no Electron, no browser overhead.

Architecture

Four-layer design. Tauri bridges native Rust performance with a modern React frontend. No Electron overhead.

1

React Frontend

React 19 + TypeScript with Tailwind CSS. 40+ components including editor panes, canvas, settings, import/export wizards. Zustand 5 for state management.

2

CodeMirror 6 Editor

30+ custom extensions: wiki-links, live preview, image preview, math rendering, Mermaid diagrams, slash commands, table editor, and Yjs collaboration binding.

3

Tauri v2 Backend

Rust-powered file I/O with path traversal protection. Collaboration server bound to localhost. Full-text search indexer. Git sync. Multi-format importers.

4

Plugin & Theme Layer

Sandboxed iframe plugin execution with restrictive CSP. 21+ themes with JSON custom theme support. i18next internationalization with 2200+ translation keys.

Getting Started

Download a pre-built binary or build from source. Available for Windows, macOS, and Linux.

build.sh 
# Clone and build
$ git clone https://github.com/Real-Fruit-Snacks/Cascade.git
$ cd Cascade

# Install dependencies
$ npm install

# Development mode
$ npm run tauri dev

# Production build
$ npm run tauri build
  Compiling cascade v0.1.0
    Finished release [optimized]
