FEATURES

Your notes, your machine

Vault-based knowledge management with live preview, wiki-links, graph view, canvas whiteboard, and real-time collaboration.

Live Preview Editor

CodeMirror 6 with three view modes — Live Preview, Source, and Reading. Syntax highlighting, code folding, bracket matching, Vim mode, LaTeX math, and Mermaid diagrams.

/heading /code /callout /table
Slash commands for quick formatting
Ctrl+P command palette (40+ commands)
21+
Built-in Themes

Wiki-Links & Graph

[[note]] links with autocomplete, backlink tracking, and heading links. Graph view visualizing connections. Tag index with browsing and renaming.

Real-Time Collab

Live collaborative editing via Yjs CRDT sync. Multiple users with live cursors and per-user colors. Auto host election with zero-config discovery.

Canvas Whiteboard

Infinite canvas with pan, zoom, and snap-to-grid. Text, file, and link cards with markdown rendering. Auto-layout algorithms. Obsidian-compatible .canvas format.

3
View Modes

Import & Export

Import from Obsidian, Notion, Bear, and Roam Research. Export to Markdown, HTML, or PDF with batch export. Deep link support via cascade:// protocol. GitHub sync for vault backup.

Obsidian Notion Bear Roam → Cascade
Cascade → Markdown HTML PDF
cascade://open/vault/note

Focus & Writing

Focus mode with paragraph dimming and typewriter mode. Word count goals with progress tracking. Split panes for side-by-side editing. Auto-save.

ARCHITECTURE

Tauri v2 architecture

React frontend with CodeMirror 6 for editing, Zustand for state, Yjs for CRDT sync. Rust backend handles file I/O, collaboration, and search.

1

React Frontend

40+ UI components with Zustand state management. CodeMirror 6 extensions for wiki-links, live preview, slash commands, and collaboration.

2

Editor Engine

CodeMirror 6 with custom extensions for wiki-links, live preview, slash commands. Yjs CRDT for real-time collaboration with WebSocket sync.

3

Rust Backend

Tauri v2 commands for file I/O, vault management, themes, plugins, and export. WebSocket collaboration server. Full-text search engine.

4

Plugin System

Sandboxed iframe execution with restrictive CSP. Built-in marketplace. 30+ configurable feature toggles. Per-plugin settings and i18next internationalization.

GETTING STARTED

Download or build

Pre-built installers for all platforms. Or build from source with Node.js 18+ and Rust 1.70+.

$ cd Cascade && npm install
