# Keyboard Shortcuts

Quick reference for Cascade keyboard shortcuts. #reference

---

## Navigation

| Shortcut        | Action                        |
|-----------------|-------------------------------|
| `Ctrl+O`        | Open file (quick switcher)    |
| `Ctrl+P`        | Open command palette          |
| `Ctrl+N`        | Create new note               |
| `Ctrl+Tab`      | Cycle through open tabs       |
| `Ctrl+W`        | Close current tab             |

---

## Editing

| Shortcut        | Action                        |
|-----------------|-------------------------------|
| `Ctrl+S`        | Save current file             |
| `Ctrl+B`        | Toggle bold                   |
| `Ctrl+Z`        | Undo                          |
| `Ctrl+Shift+Z`  | Redo                          |

---

## Search

| Shortcut        | Action                        |
|-----------------|-------------------------------|
| `Ctrl+F`        | Find in current file          |
| `Ctrl+H`        | Find and replace              |
| `Ctrl+Shift+F`  | Search across all notes       |

---

## Settings

| Shortcut        | Action                        |
|-----------------|-------------------------------|
| `Ctrl+,`        | Open settings                 |

---

## Tips

- Use `Ctrl+P` to discover any action — most features are accessible from the command palette
- `Ctrl+Shift+F` is the fastest way to find a concept spread across multiple notes
- Pair `Ctrl+O` with typed initials for instant fuzzy file switching
